Mini Preview is a simple jQuery plugin for adding live previews of to links on hover. By default, the previews were fetched as soon as the page loaded. This is great for having the previews ready ahead of time, but can eat up extra bandwidth.

See also:

  • jQuery Plugin For URL Live Preview - urlive

Basic Usage:

1. Include the jQuery library and jQuery mini preview plugin in the document.

view source
1<link href="jquery.minipreview.css" rel="stylesheet">
2<script src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.0/jquery.min.js"></script>
3<script src="jquery.minipreview.js"></script>

2. Call the plugin and you're done.

view source
1<script>
2$(function() {
3$('a').miniPreview();
4});
5</script>

3. Available options.

view source
01<script>
02$(function() {
03$('a').miniPreview({
04width: 256,
05height: 144,
06scale: .25,
07prefetch: 'pageload' // parenthover, pageload or none
08});
09});
10</script>

This awesome jQuery plugin is developed by lonekorean. For more Advanced Usages, please check the demo page or visit the official website.
